# Mirror-iOS

### 개발환경 
- 미니멈 배포버전: iOS 14.0 
- Xcode 14.3
- 서버 통신 : URL Session

### iOS 개발 명세
- 디자인 패턴: MVC 
- UI: Code Base
- 언어: Swift
- 라이브러리
  > SnapKit - 레이아웃 설정
  > VisionKit - 문서 및 이미지 OCR
