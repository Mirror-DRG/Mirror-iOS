# Mirror-iOS

### 프로젝트 설정
* 미니멈 타켓 ios 14.0
* UIkit / MVC 패턴
* URL Session

### 프로젝트 적용 프레임워크
* SnapKit - 레이아웃 설정
* VisionKit - 문서 및 이미지 OCR
